
# Islamic Quizzes Website

The purpose of this project is to create a website for people to test their Islamic knowledge. This will help them both learn new things and test what they know. 


## Features

- Quizzes that contain 15 questions each
- Buttons for redirection
- Menu


## Lessons Learned

- One of the challenges that I faced was to create a responsive website that is suitable to different devices. I overcame this by using the <meta> and also using @media in the CSS file to make it suitable to different devices.
- Another thing I need to be careful of whilst writing JavaScript code is that I need to ensure that the indentation and the brackets are properly enclosed.
